# Validador de Números: Positivo, Negativo ou Zero ➕➖

Este repositório contém um algoritmo desenvolvido em **C#** para identificar o sinal de um número inteiro ou decimal. 

## 🚀 Funcionalidades
* ✅ Entrada de dados numéricos via console.
* ✅ Verificação se o número é **Positivo** (> 0).
* ✅ Verificação se o número é **Negativo** (< 0).
* ✅ Identificação de **Zero** (Neutro).

## 🛠️ Tecnologias Utilizadas
* C#
* .NET 8.0
